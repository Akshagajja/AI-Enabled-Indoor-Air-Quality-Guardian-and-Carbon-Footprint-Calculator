# AI-Enabled-Indoor-Air-Quality-Guardian-and-Carbon-Footprint-Calculator
Developed an AI-powered Indoor Air Quality and Carbon Footprint Prediction System using Python, Machine Learning, Flask, Pandas, and NumPy. Predicted air quality levels (Good, Moderate, Unhealthy, Hazardous) and estimated annual carbon footprint from environmental and lifestyle data, achieving 96.3% prediction accuracy.
